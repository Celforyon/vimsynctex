# Installation

## With Vundle
```
Plugin 'celforyon/vimsynctex'
```
then in vim `:VundleInstall`

# Update

## Vundle
In vim `:VundleUpdate`

# Usage

## SynctexView

Opens the viewer.

## SynctexForward

Synchronises the viewer with the cursor position (file, line and column).

## SynctexUpdate

Updates variables for current buffer (e.g. try to find the PDF file).
